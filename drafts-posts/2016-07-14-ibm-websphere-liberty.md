IBM Websphere Liberty is a modern lightweight server (I have several running on my PC) that allows modern Java EE development with Java 8 and can deploy web applications as standard web archives. The last one is something I've missed when moving back to Domino. The server can be configured via XML files or an Eclipse plugin which provides editors. I've not gone beyond basic configuration so far, so I know a real-world deployment would require a lot of questions answering, possible pitfalls and potential optimisation that's required. But initial setup is very easy, easy enough to let me focus on doing the development I want. With the CrossWorlds approach, it still gives access to Domino while getting a modern HTTP server without needing an additional physical or virtual server, which would minimise expense of replacing part of the Domino stack. It's a nice environment, but may not appeal to most customers. Apache Tomcat is an obvious other server to consider, when there is time and need.
